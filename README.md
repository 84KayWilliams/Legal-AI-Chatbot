CREAC Legal AI Chatbot

AI-powered legal writing assistant that generates predictive memoranda and persuasive briefs using the CREAC structure.

## Features
- CREAC structured output
- Federal + state law analysis
- Predictive and persuasive modes
- Clean UI for demonstration

## Deployment (Render)
1. Push repo to GitHub
2. Create Web Service on Render
3. Add environment variable:
   OPENAI_API_KEY=your_key
4. Deploy

## Local Run
pip install -r requirements.txt
python app.py
